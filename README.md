# ToCoE - Table of Contents of Everything

A platform for producing CC0 tables of contents (ToCs) of written works and associating them with subject headings, all through volunteer labor and linked open data.

See http://aboutness.org

== Supported book/text sources ==
Currently, ToCoE can help users create ToCs from:
* scans in the Internet Archive's Open Library (using its public API)
* texts in Project Gutenberg (requiring an accessible instance of the Gutendex API server [not included])

== Supported subject heading sources ==
Currently, ToCoE supports drawing subject headings from:
* the Library of Congress (LCSH) 
* Wikidata
